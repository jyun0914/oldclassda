環境設定（cont.）

  • Java 環境變數設定（Windows）

  • 我的電腦右鍵->內容->進階->環境變數 新增:
    • a. 變數名稱 PATH
      變數值 C:\Program Files\Java\jdk1.7.0\bin

    • b. 變數名稱 CLASSPATH
      變數值 C:\Program Files\Java\jdk1.7.0\lib

    • 如果有別的環境變數，則要在前面加上分號
